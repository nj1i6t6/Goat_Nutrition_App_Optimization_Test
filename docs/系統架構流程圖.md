## 數據流程圖

```mermaid
sequenceDiagram
    participant U as 使用者
    participant F as 前端 Vue.js
    participant A as 後端 API
    participant DB as 資料庫
    participant AI as Gemini API
    
    U->>F: 登入系統
    F->>A: POST /api/auth/login
    A->>DB: 驗證用戶資料
    DB-->>A: 返回用戶資訊
    A-->>F: 返回認證狀態
    F-->>U: 顯示主界面
    
    U->>F: 查看山羊列表
    F->>A: GET /api/sheep/
    A->>DB: 查詢山羊數據
    DB-->>A: 返回山羊列表
    A-->>F: 返回 JSON 數據
    F-->>U: 顯示山羊表格
    
    U->>F: 請求 AI 營養建議
    F->>A: POST /api/agent/recommendation
    A->>AI: 呼叫 Gemini API
    AI-->>A: 返回 AI 建議
    A->>DB: 儲存建議記錄
    A-->>F: 返回營養建議
    F-->>U: 顯示個性化建議
    
    U->>F: 上傳 Excel 檔案
    F->>A: POST /api/data/import
    A->>A: 解析 Excel 數據
    A->>DB: 批量插入數據
    DB-->>A: 確認插入完成
    A-->>F: 返回匯入結果
    F-->>U: 顯示匯入報告
```

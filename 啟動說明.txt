最終的專案啟動指南（生產模式）
這是在您功能完善後，準備公開給他人使用的標準啟動流程。
前提：
電腦已安裝 Python, Node.js, PostgreSQL。
PostgreSQL 服務正在運行。
在 VSCode 中打開專案根目錄。
第 1 步：前端打包 (只需在修改前端程式碼後執行一次)
打開一個終端機。
執行 cd frontend
執行 npm run build，並等待其完成。
第 2 步：啟動統一後端伺服器 (一個終端機)
打開一個新的終端機。
執行 cd backend
執行 .\.venv\Scripts\Activate.ps1
執行 waitress-serve --host=0.0.0.0 --port=5001 run:app
保持此終端機運行。
第 3 步：使用 ngrok 公開服務 (另一個終端機)
打開一個新的終端機。
執行 ngrok http 5001 (指向我們的後端伺服器)。
將 ngrok 生成的 https://... 網址分享給他人即可訪問。
